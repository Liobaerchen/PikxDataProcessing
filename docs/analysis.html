Is extreme heat related to increased violence against women?

Stehen Hitzeextreme mit Gewalt gegen Frauen in Zusammenhang?

Model 2

DVrct = β · Heatrct + γ · Rainrct + δc + τt + εrct
  • FE: Country + year
  • Controls: Rainfall, clustered SEs
  • Variation: Within-country, across regions
  • Question: Are hotter regions within a country more violent?

Model 3 — binomial check

For each region, check if the hotter year is also the more violent year:

  • 2 years -> score 1 or 0 directly
  • 3+ years -> check all pairs; majority concordant = 1, discordant = 0, ties dropped

Count the 1s across all regions and locate that in a Bernoulli(p = 0.5) distribution. That's the test.

n: 142 single-year regions excluded; 185 regions and 594 observations retained.

Note: Model 2 controls for year effects; the binomial test cannot. If both point in the same direction, that is reassuring.

Assumption Checks for Model 2

Linearity and Heteroscedasticity

AIC linear vs. quadratic

  • Base model AIC: 4926.893269703587,
  • Square model AIC: 4927.55049744022
The quadratic term doesn't add anything. This indicates it makes sense to use the linear model! Good.

Notes

Plot 1

  • No obvious linear relationship is visible, but also no e.g. exponential relationship.
  • Mean of the residuals is roughly around 0 across the range, so linearity looks okay; there's no obvious curve.
  • The spread of residuals is wider in the middle. That funnel-ish shape suggests some heteroscedasticity.
  • Heteroscedasticity is present, but I think the clustered SEs make the inference robust to this!

Plot 2

  • Heat only takes a few distinct values, which makes sense because it's a share of 12 months, so it can only be multiples of 1/12. This is an outcome of the variable construction, not really a problem. But it means heat is almost ordinal, very crude, which should maybe be mentioned later.
  • There is no immediately visible linear trend. This could mean one of several things:
    • There is no linear relationship
    • The relationship is obscured by the effects that I will control for
    • The relationship is very weak (probably true)
      • So, this plot isn't alarming. Linearity could be given. This result is actually kind of expected; without fixed effects, I am comparing different countries and different years, probably obscured by many other variables that I am not controlling for.

Skew of DV

Notes

  • The DV is right-skewed (which is expected for this variable).
  • However, OLS does not assume the DV itself is normally distributed, only that the residuals are, and that assumption relaxes with large n due to the CLT.
  • The residuals were already checked and looked okay.
  • Skew could be an issue if it leads to predictions below 0, which would be nonsensical. But most values are between 5–30%, so out-of-bounds predictions are unlikely.

Since both are measured in percent, a 1 increase in heat and DV doesn't make sense.

BUT:
A 10 percentage-point increase in extreme-heat-month count is associated with a roughly 0.8 percentage-point increase in DV prevalence (controlling for rainfall, country, and year fixed effects etc. etc.).

This seems small, but it is meaningful.

Interpretation (population perspective)

This seems small, but it is meaningful if you consider
Even if the increase in risk for each person is tiny, multiplying that small increase over a large population can produce a huge number of additional cases overall.

Imagine:

  • 1,000,000 women experience DV in these countries
  • 0.8% increase -> ~8,000 additional cases

The point is explained here: The Axiom of Rose (Malik, 2006)

MODEL 3: binomial check

For each region, check if the hotter year is also the more violent year:

  • 2 years -> score 1 or 0 directly
  • 3+ years -> check all pairs; majority concordant = 1, discordant = 0, ties dropped

Count the 1s across all regions and locate that in a Bernoulli(p = 0.5) distribution. That's the test.

RESULTS

Regions included: 109
Hotter = more violent: 51
Ties: 76
Single-year regions: 142
One-tailed p-value: 0.778

Basically: 50/50 split.

BUT:

This might simply reflect coarse aggregation and low within-region variation.

Interpretation

The result suggests no clear directional pattern at this level of aggregation.

Possible explanations:

  • Effect only exists at short time scales (not yearly averages)
  • Measurement error in both DV and heat
  • Not enough within-region variation
  • Cross-sectional confounding still dominates in Model 2
  • Omitted variables correlated with heat

Alternative Explanations & Assumption Checks (Model 3)

Possible Explanations for Null Result

may be because of coarseness of measure

possible explanations:

  • No real effect at the time scale I use / aggregation is too coarse
    Heat might only affect violence in the very short term (like during very hot days), or on very long time scales (how to test that?) but my data averages everything over a whole year. And I use the “share of extreme months” over a year. This turns a ratio scale into binary, ... and ignores a lot of data.
  • Measurement error
    My DV data comes from surveys, and people may not report violence accurately. There are also influences like culture, politics, etc. etc. My heat variable is also smoothed over many months. When both variables are noisy, real effects get blurred and look like zero. And the binomial test is not very powerful at all.
  • Not enough change within regions
    Within each region, heat does not change that much from year to year. If nothing really changes, the model cannot detect any effect. Maybe average heat differences are just too small. NOTE: try a model that only includes large differences!
  • Cross-sectional bias in Model 2 (it's actually a 0 effect)
    Hotter regions are often also poorer or more rural. These factors could impact DV. So it may look like heat causes violence, but it’s actually these other differences. Also, different regions may have different norms about gender and violence. These differences don’t change much over time, but they can create strong differences between regions.
  • Similarly: Omitted variables (again, it may really be a null effect)
    There may be other things affecting violence (like economic stress or conflict) that I did not include in the model. If those are linked to heat, results can be wrong.

Mechanism Check 1: Within-Region Heat Variation

How much does heat actually vary within each region over time?

If most regions cluster near 0, that supports this alternative explanation (they do).

Mechanism Check 2: Does stronger variation change results?

- If p shrinks as threshold rises, null result probably a power-problem (it does!).
- If p stays ~0.5 regardless -> the null result is probably robust.

Mechanism Check 3: Power of the Binomial Test

Power = probability of detecting a true effect. A weak true effect (55%) would probably go undetected with n = 109. So the null result is not strong evidence against an effect. The test just may not be sensitive enough with n = 109.
